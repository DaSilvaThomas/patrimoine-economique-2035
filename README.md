# Patrimoine économique français – 2035

Application web interactive de type scrollytelling présentant une vision prospective du patrimoine économique français à l’horizon 2035.

## Prérequis

* Node.js (version 18 ou supérieure recommandée)
* npm

## Installation

Cloner le dépôt :

```bash
git clone https://github.com/DaSilvaThomas/patrimoine-economique-2035.git
cd patrimoine-economique-2035
```

Installer les dépendances :

```bash
npm install
```

## Lancer le projet en local

```bash
npm run dev
```

L’application est ensuite accessible à l’adresse suivante :

```
http://localhost:3000
```

## Notes

Ce projet est un support de présentation. Il ne nécessite aucune base de données et fonctionne entièrement côté client.
